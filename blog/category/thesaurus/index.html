Atramhasis v0.6.0 released

  • 24 augustus 2017
  • Koen van Daele
  • English
  • releases, thesaurus, open source
  • atramhasis

A new version (0.6.0) of :pypi:`Atramhasis <atramhasis>` has been released. Atramhasis is an open-source, web-based SKOS editor. You can use it to browse SKOS vocabularies, thesauri, authority files, word lists, … Atramhasis comes with an admin interface you can use to add the concepts and collections of your vocabularies. To make integrating vocabularies with your other IT systems easy, all data in Atramhasis is exposed through a read an write REST webservice.
